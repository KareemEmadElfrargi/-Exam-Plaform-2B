# 2B School Exam Platform

This is a React-based exam platform developed for 2B School. It includes:

- **Exam Interface**: A timed exam with multiple-choice questions.
- **Security Features**: Anti-cheating measures like blur on focus loss and full-screen enforcement.
- **Results**: Immediate grading with gender-personalized feedback.
- **Tech Stack**: React, TypeScript, Vite.

## How to Run

1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local server.

## Deployment

This project is ready for deployment on Vercel.
